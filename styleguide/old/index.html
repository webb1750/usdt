Grid Layout

The USDT Grid Layout is based on the Flexbox model and consists of containers, rows, and columns to create a 12 column grid layout structure.

Please note:

  • When using this grid layout, content must be placed within columns and only columns may be immediate children of rows.
  • You can nest grid rows, but you must nest them within a column.
  • Containers provide a means to horizontally center and add padding to your page. A grid container (.usdt-grid-container) should be used as the main parent (wrapping) element. There is also a full width container (.usdt-grid-container-fluid) that streches 100% width of the page.
