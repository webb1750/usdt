Grid Layout

The USDT Grid Layout is based on the Flexbox model and doesn't use floats. For easy page layouts, you can use the usdt-auto-grid class to automatically handle the width for each column.

For example, if you want a three column layout and each column to have the same width, you simply place three HTML elements with the class of usdt-col inside a wrapping usdt-auto-grid element to achieve that layout.
